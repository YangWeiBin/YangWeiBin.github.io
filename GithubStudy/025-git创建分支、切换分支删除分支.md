# 025 git分支操作      
---
## 基本操作  
1. 创建分支  
	- git checkout -b gh-pages  
2. 切换分支   
	- git checkout gh-pages   
3. 先切换其他分支，才能删除此分支   	  
	- git checkout master  
	- git branch -d gh-pages   

## 总结  
1. 分支类似于一个版本  
2. 提交时，所在不同的分支提交到github仓库的不同分支
